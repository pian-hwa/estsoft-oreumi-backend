# HTML 문서 작성 태그 정리 (2025-11-13)

- ## 정보
    - 과목 : HTML
    - 난이도 : 0.5 / 5.0
    - 메모 : HTML의 기본

- ## 기본적인 구조
    ```html
    <!DOCTYPE html> <!-- HTML 버전을 지정한다. -->
    <html lang="ko"> <!-- lang 속성으로 주 언어를 명시한다. -->
    <head> <!-- meta, title, link, style, script 태그를 작성한다. -->
        <meta charset="UTF-8"> <!-- charset 속성으로 인크립 방식을 지정한다. [필수] -->
        <title>제목</title> <!-- HTML 파일의 제목을 설정한다. [필수] -->
        <link> <!-- 외부 CSS 파일을 가져온다. -->
        <style></style> <!-- css를 작성하여 스타일을 부여한다. -->
        <script></script><!-- js을 실행시키거나, 외부 js 파일을 가져온다.-->
    </head>
    <body> <!-- viewport에 보이는 콘텐츠를 작성한다. -->

        <!-- 콘텐츠 -->

    </body>
    </html>
    ```

- ## 문서 작성 태그 모음

    - ```<p>``` : 한 문단을 생성한다. 내부에는 인라인 요소들이 들어간다.
    - ```<h1>``` : h1 ~ h6, 제목을 생성한다. 숫자가 커질수록 글자 크기가 작아진다.
    - ```<a>``` : 하이퍼 링크를 생성한다. ```href``` 속성에 따라 달라진다.
        - ```URL``` : 외부 문서로 이동한다.
        - ```#``` : 문서의 최상단으로 스크롤하여 올라간다.
            - ```#``` 옆에 id 속성을 지정해둔 요소의 id를 입력하여 해당 요소로 스크롤하여 올라간다.
    - ```<br>``` : 강제로 개행한다.
    - ```<hr>``` : 문단에 양 끝을 잇는 구분자를 생성한다.
